### 属性

|                属性名                 | 说明                         | 类型                | 默认值  |
| :-----------------------------------: | ---------------------------- | ------------------- | ------- |
|                 type                  | Tag 的类型                   | string              | -       |

| `primary/success/info/warning/danger` | 不同类型匹配到的数组或字符串 | string/number/array | -       |
