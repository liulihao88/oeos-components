# oeos-v3-components的开发计划

> 组件开发的原则, 是以项目为第一优先级. 一旦项目中实际需要某一个组件, 会提前或跟随项目进度一起开发这个组件. 以使用频率为第二优先级. 还有如果其他项目有比较优秀的组件, 我也会借鉴然后进行开发, 这时这个组件会被优先开发出来.

### 1. 下一阶段计划开发的组件, 优先级, 由高到低(5月30日前初步完成)

1. table组件
2. form组件
3. description组件
4. 其他组件
