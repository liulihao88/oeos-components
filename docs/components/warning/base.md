### 属性

|     属性名     | 说明                                | 类型            | 默认值 |
| :------------: | ----------------------------------- | --------------- | ------ |
|      type      | options参数的类型, `icon/info/''` | string          | ''     |
|    content     | 显示的正文内容                      | String          | -     |
|    width     | 宽度                      | String/Number          | 100%     |
|    dangerouslyUseHTMLString     | 是否将 `content` 作为 HTML 片段处理                      | boolean         | false     |
