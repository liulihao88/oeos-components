### 属性

|   属性名    | 说明                          | 类型   | 默认值 |
| :---------: | ----------------------------- | ------ | ------ |
|   options   | 选项                          | Array  | -      |
|   column    | 一行 Descriptions Item 的数量 | number | 3    |
| label-width | label的宽度                   | string | ''     |
|    title    | 标题文本，显示在左上方        | string | ''     |
|    extra    | 操作区文本，显示在右上方      | string | ''     |
|    showAll    | 是否显示全部文本      | boolean | false     |
